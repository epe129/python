# Python Projects Collection

This repository contains various Python projects and exercises covering different programming concepts and applications.

## Project Categories

### Games
- **Seikkailupeli**: A 2D adventure game built with Pygame where players can explore rooms and collect items
- **MadLibs Game**: Interactive story game where players fill in words
- **Betting Game**: Simple betting simulation game
- **Bingo**: Basic bingo game implementation

### Utilities
- **Todo Apps**: Multiple todo list implementations
  - Class-based implementation with file storage
  - Simple command-line interface
- **Budget Tracker**: Personal finance tracking application
- **Text Summarizer**: Uses transformers to summarize text documents
- **Random Jokes**: Joke generator supporting both Finnish and English jokes

### Learning Projects
- **Custom Programming Language**: Basic implementation of a custom programming language
- **Chat Bot**: Simple conversational bot
- **Calculator**: Basic calculator implementation
- **Adventure Story**: Text-based adventure game

### GUI Applications
- **Diary**: Simple diary application with Tkinter
- **Feet to Meters**: Unit conversion tool with GUI
- **Timer**: Basic timer application

## Installation

Requires Python 3.8 or newer. Install dependencies for specific projects using:

```sh
pip install -r requirements.txt
```

## Usage

Each project is contained in its own directory. Navigate to the project directory and run the main Python file:

```sh
cd project_directory
python main.py
```

## Key Features

- Multiple standalone applications
- Mix of CLI and GUI applications
- File handling and data persistence
- Game development examples
- Text processing utilities

## Testing

Test files are located in the `testit/` directory. Run tests using:

```sh
python -m pytest testit/
```
